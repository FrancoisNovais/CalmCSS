## Introduction

CalmCSS is a classless CSS theme designed as a starting point for web development.  
It provides a clean typographic base, a coherent semantic structure, automatic dark mode, and a modular organization (variables, reset, base).  
Ideal for quickly starting a project without worrying about styling, while maintaining a sober and professional look.

---

## Main Goals

- **Ready to use**  
  Just write your HTML, it works. No need to think about styles.

- **No extras**  
  No classes, no imposed components. Just the essentials.

- **Pure HTML**  
  Semantic code, no syntax to learn.

- **Naturally responsive**  
  Everything adapts, no setup needed.

- **Subtle, readable, elegant style**  
  A neutral, clear base that highlights your content.

- **Solid compatibility**  
  Tested on major modern browsers.
